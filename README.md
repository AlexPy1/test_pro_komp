# API 
Для получения всех имеющихся файлов отправляем GET запрос на api/files
```
[
    {
        "columns": [
            "Game Number",
            " \"Game Length\""
        ],
        "file": "2.csv"
    },
    {
        "columns": [
            "Month",
            " \"Average\"",
            " \"2005\"",
        ],
        "file": "sample1.csv"
    },
    {
        "columns": [
            "Game Number",
            " \"Game Length\""
        ],
        "file": "sample4.csv"
    }
]
```
Для получения информации из файла отплавляем GET запрос на /api/<name_file>
